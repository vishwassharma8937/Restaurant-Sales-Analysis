# Restaurant Sales & Labor Cost Analysis

## Project Overview

This project analyzes restaurant sales and labor cost data using Python.

The project demonstrates:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- KPI Calculation
- Data Visualization
- Business Insights

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## KPIs

- Total Sales
- Total Profit
- Labor Cost %
- Food Cost %
- Profit %
- Average Bill

## Visualizations

- Store-wise Sales
- Store-wise Profit
- Labor Cost Pie Chart
- Sales Trend
- Sales Distribution
- Sales vs Profit

## Files

- restaurant_sales_analysis.ipynb
- restaurant_sales.csv
- restaurant_sales_cleaned.csv
- store_performance.csv
- summary_report.csv
